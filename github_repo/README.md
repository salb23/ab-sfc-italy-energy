# AB-SFC Italy — Energy Policy Framework

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/)
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.PLACEHOLDER.svg)](https://doi.org/10.5281/zenodo.PLACEHOLDER)

An **Agent-Based Stock-Flow Consistent (AB-SFC)** macroeconomic framework for the
quantitative analysis of energy-price shocks, decarbonisation policy and the
distributional incidence of the low-carbon transition in the Italian economy.

This repository contains the replication code for the paper *"An Agent-Based
Stock-Flow Consistent Framework for Energy Policy Design"* (SIE 2026 conference
submission).

---

## Headline findings

The framework is calibrated on the Italian economy (HFCS 2017, EU-SILC 2019,
ENTSO-E, GME, TTF) and used to evaluate fourteen policy scenarios under a
calibrated 2022-type gas-price shock, with N = 100 seeds per cell under common
random numbers (CRN). Three substantive results emerge:

1. **Diversified generation stack + moderate corrective carbon price (τ = 85
   €/tCO₂)** dominates the no-policy shock baseline jointly on emissions
   (−5,150 tCO₂, *p* < 10⁻⁴) and on the four-dimensional Resilience Index
   (+0.067 points, *p* < 10⁻⁴), with a much smaller peak-price excursion than
   any binding-quantity alternative.

2. **Weitzman price-vs-quantity asymmetry** is endogenously reproduced.
   Tight ETS caps drive the allowance price to its ceiling and push the
   wholesale electricity price to 940 €/MWh — an order of magnitude above
   baseline — collapsing the RESI composite to 0.39, even though they
   achieve the largest emissions reductions in the grid (−57.5%).

3. **High-efficiency appliance adoption** is the primary household-side
   adjustment margin. H-class adoption rises from 3.3% in the no-shock
   steady state to 43.2% under the gas shock alone, to 63.7% under
   τ = 85, and saturates at 95% under the tightest ETS cap. Policy
   instruments amplify, rather than create, this adaptive response.

## Reproducing the production grid

### Install (Python 3.10+)

```bash
git clone https://github.com/salb23/sie2026-ab-sfc-italy-energy.git
cd sie2026-ab-sfc-italy-energy
python -m venv .venv && source .venv/bin/activate   # optional but recommended
pip install -r requirements.txt
```

### Run the production grid

```bash
jupyter nbconvert --to notebook --execute notebooks/SIE2026_production.ipynb \
                  --output /tmp/executed.ipynb
```

This regenerates every CSV, every PNG and every LaTeX table in
`outputs_production/`. Random seeds are fixed (`SEED_BASE = 20260101` with
common random numbers across scenarios), so the numerical output is bit-for-bit
identical to what is reported in the paper.

**Expected runtime:** ~3 minutes on 4 cores (1,400 seed-runs in total).
**Configuration knobs** are at the top of the notebook: `N_SEEDS = 100`,
`N_WORKERS = 4`. Setting `parallel = False` gives a serial run for debugging.

### Run the SFC closure tests

```bash
cd model_full
python -m pytest tests/ -v
```

The tests verify that the Transaction Flow Matrix and Balance Sheet Matrix
balance to floating-point tolerance at every simulation step, for every phase
of the model.

## Methodological notes

**Calibration sources** — All parameter values are anchored to publicly
available data: HFCS 2017 (quintile deposits), EU-SILC 2019 (energy
expenditure shares, credit access η_κ), Istat ICP (CPI baseline), ENTSO-E
(generation by fuel), GME MGP (zonal day-ahead prices), TTF (gas marginal
cost), Eurostat NRG_T1_11 (national energy balance), and Eurostat SBS (firm
size distribution). No proprietary or subscription-gated dataset is required.

**Stock-flow consistency** — Every flow appears in the Transaction Flow Matrix
and every stock in the Balance Sheet Matrix, with row and column sums
identically zero by construction. The redundant equation is computed as a
residual at every time step and stored as a numerical consistency diagnostic.

**Statistical inference** — Common random numbers across the fourteen
scenario cells (SEED_BASE = 20260101) eliminate cross-cell sampling noise.
Pairwise differences are tested with the paired Wilcoxon signed-rank test
(implemented in-house, no SciPy dependency); 90% bootstrap percentile CIs are
reported on every aggregate KPI (2,000 resamples).

**Sensitivity** — A one-at-a-time sweep over the four principal auxiliary
behavioural parameters (χ, φ_pm, κ_invest, ρ_peer) anchored to the
italy_baseline_2022_shock cell shows that the policy ranking established in
the production grid is robust to the calibration margins on which the AB-SFC
literature has not yet converged.
